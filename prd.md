# **3. Section 1 – 문제 인식 (The Problem)**

### **섹션 제목**

**중학생 수행평가, 왜 이렇게 어렵게 전달될까?**

### **내용**

* 수행평가 공지 방식이 여전히 **A4 출력물 + 단톡방 사진**에 의존
* 친구가 사진 안 올리면 일정 누락
* 변경 사항 전달 어려움
* 준비물 오류·마감일 혼동
* 학부모는 정확한 정보를 얻기 어려움

### **비주얼**

* 칠판의 A4 공지 → 학생이 폰으로 찍는 모습
* 카톡 단톡방에서 메시지가 섞여 내려가는 화면

---

# **4. Section 2 – 솔루션 소개 (Our Solution)**

### **섹션 제목**

**무한수행으로 모든 일정이 한 번에 해결됩니다.**

### **내용 포인트**

* 교사가 올린 수행평가 공지가
  → 학생 캘린더에 자동 등록
  → 알림으로 전달
  → 학부모도 동일하게 확인 가능

### **하이라이트 기능**

* 자동 캘린더 생성
* D-Day 알림
* 준비물 체크
* 일정 변경 즉시 안내
* 반별/과목별 공지 자동 분류

# **6. Section 4 – 기대 효과 (Impact)**

### **섹션 제목**

**수행평가 누락률 80% ↓, 학생 부담 70% ↓**

### **하이라이트**

* 일정 누락률 **50% → 10% 이하 감소**
* 학부모 정보 불균형 해소
* 교사 업무량 감소
* 학생 자기주도 학습 증가
* 학교 전체의 학사 관리 효율화

### **그래픽**

* Before(50%) → After(10%) 그래프
* 교사·학생·학부모 3자 이득 구조 다이어그램
